# Power BI – Análise de Vendas por diferentes critérios
Dashboard em Power BI com análise de vendas e lucro por país. O projeto explora dados de vendas permitindo a visualização geográfica, comparações entre países e análise de indicadores financeiros como vendas e lucro

## Descrição
Dashboard desenvolvido em Power BI para análise de vendas e lucros considerando produto, segmento, mês e país. 

## Funcionalidades
- Mapa interativo por país
- Análise de Sales e Profit
- Gráficos comparativos entre produtos, segmento e país

## Observações
Os filtros e segmentações funcionam ao abrir o arquivo no Power BI Desktop.
